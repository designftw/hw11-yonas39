Halfway Submission Report

For the halfway point milestone, I have implemented the Google Cloud API translation in my code using the Axios library. This implementation allows users to select a target language from a list of options. Then it translates messages from the source language to the target language using the Google Translate API. I obtained an API key to access the Google Translate API and stored it in a constant variable called API_KEY. This API key is required to request the API and receive translation responses.

In my code, I defined a method called translateMessage that takes message content as an argument and sends a POST request to the Google Translate API with the message content and the target language. The response from the API is then used to update the translatedText property in the data object.

I also defined another method called translateAllMessages that iterates over all messages and uses the translateMessage method to translate each message. This method is called when the user selects a new target language from the dropdown list.
